# 高阶函数

## 什么是高阶函数

- 高阶函数是对其他函数进行操作的函数, 可以将它们作为参数或返回它们
- 简单说, 告诫函数是一个函数, 接受函数作为参数或者将函数作为输出返回

## map/reduce/filter

map 遍历生成新数组
reduce 累加
filter 过滤
flat 将多维数组拉平为一维数组

```js
let arr = [1, 2, 3, [4, 5, [6, 7]]];
let arr1 = arr.flat(3); // 将三维数组转换为一维数组
let arr2 = arr.flat(Infinnity); // 未知多少层数组, 都转化为一维数组
```
